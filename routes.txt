Route-array format:
array(
  'url' => an actual URL (should be absolute)
  'path' => a path array e.g. array('posts', '23') : /posts/23
  'query' => query array e.g. array('p' => '27') : ?p=27
  'mergeQuery' => boolean, whether to merge with current query
  'fragment' => fragment string e.g. 'bottom' : #bottom
  'controller' => Controller-object or name (w/ or w/o 'Controller'-suffix)
  'action' => name of action as string
  'parameters' => array of parameters e.g. array(23, 1)
)
if 'url' is set, the url is returned
if 'controller' isn't set, 'controller', 'action' and 'parameters' default to their current values
if 'controller' is set, 'action' defaults to 'index' and 'parameters' defaults to array()
if 'path' is set, a link based on 'path', 'query' and 'fragment' is returned
if 'query' isn't set and 'controller', 'action' and 'parameters' are left unchanged, the 'query' is set to the current query
Some examples:
 - array()
    returns a link to the current page
 - array('fragment' => 'test')
    returns a link to the current page + the fragment #test
 - array('controller' => 'Pages')
    returns a link to the index-action of the PagesController
Other legal values are:
 - an object implementing ILinkable, in which case ->getRoute() is called
 - null, in which case a link to the frontpage is returned
 - a string, in which case the string is returned (OR NEW FORMAT!!)
